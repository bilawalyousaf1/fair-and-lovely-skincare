# Fair & Lovely Skincare

A modern, fully responsive static landing page for a skincare brand. The website showcases featured products, customer testimonials, and brand benefits with smooth animations and an interactive dark mode toggle.

## Features

- Hero section with product imagery and call‑to‑action buttons
- Features section highlighting key product benefits
- Product grid displaying cards with images, ratings, and prices
- Customer testimonials section
- Newsletter signup form
- Footer with contact information and social media links
- Interactive cursor follower effect
- Image hover effects (product image swapping)
- Light / dark theme toggle with persistent icon change
- Fully responsive layout for mobile and desktop devices

## Technologies Used

- HTML5 (Semantic markup)
- CSS3 (Grid, Flexbox, Animations, CSS Variables, Media Queries)
- JavaScript (ES6 Modules)
- [Font Awesome](https://fontawesome.com/) – Icons
- [Google Fonts](https://fonts.google.com/) – Poppins and Playfair Display
